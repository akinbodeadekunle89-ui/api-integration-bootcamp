# BrightRetail Customer API Investigation

## Overview

This project documents the investigation and testing of the BrightRetail Customer API.

The objective was to understand the API structure, endpoints, authentication approach, request methods, and response formats before beginning enterprise integration.

---

## Technologies

- REST API
- HTTP
- JSON
- Postman
- GitHub

---

## Tested Endpoints

- GET /users
- GET /users/{id}
- GET /posts
- POST /posts
- PUT /posts/{id}
- PATCH /posts/{id}
- DELETE /posts/{id}
